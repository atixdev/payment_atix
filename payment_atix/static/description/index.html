Instalación y Configuración del Módulo Pasarela de Pago Atix

Requisitos

  • Instancia de Odoo Community o Enterprise
  • Alojamiento “on premise” (Local, en la nube) u “odoo.sh”
  • Tener la aplicación Proveedor de pago: ATIX dentro de la lista de aplicaciones del sistema
  • API KEY (Soles y/o dólares) de comercio:
    • Obten el API KEY en https://dashboard.atix.com.pe/login

Pasos para la Configuración

  1. Ir a Aplicaciones e instalar el módulo Proveedor de pago: ATIX.
  2. Ir a Sitio Web -> Configuración -> Proveedores de Pago y activar Proveedor de pago: ATIX.
  3. Habilitar el proveedor de pago y establecer los API KEY en Dólares y/o Soles. El módulo funciona tanto si es solo en soles, solo en dólares, o ambos establecidos a la vez.
  4. Si las API KEY son de producción, cambia el estado a "Habilitado". Si son de pruebas, cambia el estado a "Modo de prueba".
  5. Copiar la url de redireccionamiento generada en Odoo y pegarla en los campos Redirección si la transacción es aprobada y Redirección si la transacción es rechazada en el dashboard de Atix.

Pruebas

Para verificar el funcionamiento del método de pago, sigue estos pasos:

  1. Accede al flujo de compra del usuario en tu sitio web.
  2. Selecciona el producto que deseas comprar y procede al pago.
  3. Verifica que el método de pago de Atix esté disponible y seleccionado por defecto si es el único método de pago.

Una vez seleccionado el método de pago, completa la transacción y verifica que se procese correctamente.

Si la transacción es exitosa, deberías ver una confirmación en pantalla. Si hay algún problema, revisa los logs y la configuración del módulo.
